# 알고리즘 문제 풀이

- 학습 플랫폼

  - [프로그래머스](https://programmers.co.kr/)
  - [리트코드](https://leetcode.com/)
  - [백준](https://www.acmicpc.net/)

- 학습 루틴

  1. 학습 플랫폼에서 문제를 2개 선정하여 1시간 동안 푼다.

  2. 문제 당 30분 내에 풀지 못할 경우, 다른 사람의 풀이 방법을 보고 이해한 후 다시 풀어본다.

  3. [템플릿](https://github.com/dawwson/algorithm/blob/main/template.md)에 맞춰 내용을 정리한다.

- 사용 언어

  - 1순위: `Java`

    - 주력 언어에 좀 더 집중하기 위함.

  - 2순위: `JavaScript` / `TypeScript`
