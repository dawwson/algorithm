### Intuition

---

<!--
- 제약 조건
- 단순 접근의 한계
- 필요한 성질 (시간/공간/구조)
- 선택한 해법

- 제약 조건:
  - 입력값의 개수 `N`은 최대 `500,000`이며, 입력되는 정수의 절대값은 `4,000` 이하이다. 시간 제한이 `2`초이므로 `O(N log N)` 알고리즘으로 충분히 해결 가능하다.

- 단순 접근의 한계:
  - 산술평균 계산 시 `Math.round`를 활용한 반올림 처리와 최빈값이 여러 개일 때 '두 번째로 작은 값'을 선택해야 하는 조건이 까다롭다.

- 선택한 해법:
  - 각 통계값을 구하는 로직을 독립적인 함수로 분리하여 설계했다.
  - 중앙값과 범위를 구하기 위해 정렬을 활용하며, 최빈값은 `HashMap`과 `Stream API`를 조합해 빈도수를 관리한다.

<br>

### Approach

---

<!--
- 실제로 채택한 알고리즘
- 자료구조 선택 이유
- 단계별 처리 방식
-->

- 자료구조 선택:
  - 빈도수 기록을 위해 HashMap`<Integer, Integer>`를 사용했다.

  - 최빈값 후보군을 관리하기 위해 `List<Integer>`를 생성하고 `Stream`으로 필터링 및 정렬을 수행했다.

- 단계별 처리 방식:
  - 산술평균: 모든 합계를 `double` 타입으로 구한 뒤 `Math.round()`로 반올림하여 반환한다.

  - 중앙값: 배열을 복사한 후 오름차순으로 정렬하여 중간 지점(`N/2`)의 값을 추출한다.

  - 최빈값: `map.getOrDefault`로 빈도수를 체크하고 `Collections.max()`로 최대 빈도수를 찾는다. 이후 해당 빈도수를 가진 키값들을 모아 정렬한 뒤 조건에 따라 값을 반환한다.

  - 범위: 정렬된 배열의 마지막 요소와 첫 번째 요소의 차이를 계산한다.

<br>

### Complexity

---

- Time complexity: `O(N log N)`
  - 중앙값과 범위 계산 단계에서 평균 시간복잡도가 O(N log N)인 `Arrays.sort()`가 수행된다.

- Space complexity: `O(N)`
  - 입력 데이터 저장을 위한 배열과 빈도수 관리를 위한 `HashMap` 사용으로 인해 입력 크기에 비례하는 공간이 필요하다.

<br>

### Code (Java)

---

```java
import java.util.*;
import java.util.stream.Collectors;
import java.lang.*;
import java.io.*;

class Main {
    public static void main(String[] args) throws Exception {

        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());
        int[] numbers = new int[N];

        for (int i = 0; i < N; i++) {
            numbers[i] = Integer.parseInt(br.readLine());
        }

        System.out.println(calculateAvg(numbers));
        System.out.println(calculateMedian(numbers));
        System.out.println(calculateMode(numbers));
        System.out.println(calculateRange(numbers));
    }

    public static long calculateAvg(int[] numbers) {
        double sum = 0.0;

        for (int number : Arrays.copyOf(numbers, numbers.length)) {
            sum += number;
        }

        return Math.round(sum / numbers.length);
    }

    public static int calculateMedian(int[] numbers) {
        int[] copy = Arrays.copyOf(numbers, numbers.length);

        Arrays.sort(copy);

        return copy[numbers.length / 2];
    }

    public static int calculateMode(int[] numbers) {
        Map<Integer, Integer> map = new HashMap<>();

        for (int number : Arrays.copyOf(numbers, numbers.length)) {
            map.put(number, map.getOrDefault(number, 0) + 1);
        }

        int maxFreq = Collections.max(map.values());

        // 최대 빈도수를 가진 숫자들만 뽑아서 리스트화
        List<Integer> modes = map.entrySet().stream()
                .filter(entry -> entry.getValue() == maxFreq)
                .map(Map.Entry::getKey)
                .sorted() // 오름차순 정렬
                .collect(Collectors.toList());

        // 최빈값이 여러 개면 두 번째 값, 아니면 첫 번째 값
        return (modes.size() > 1) ? modes.get(1) : modes.get(0);
    }

    public static int calculateRange(int[] numbers) {
        int[] copy = Arrays.copyOf(numbers, numbers.length);

        Arrays.sort(copy);

        return copy[copy.length - 1] - copy[0];
    }
}
```
