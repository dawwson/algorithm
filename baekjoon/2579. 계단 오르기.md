> 문제 제목 : 2579. 계단 오르기

### Intuition

---

<!--
- 제약 조건
- 단순 접근의 한계
- 필요한 성질 (시간/공간/구조)
- 선택한 해법
-->

- 제약 조건:
  - 한 번에 한 계단 또는 두 계단씩 오를 수 있으며, 연속된 세 개의 계단을 모두 밟아서는 안 된다.
  - 또한, 마지막 도착 계단은 반드시 밟아야 한다.

- 단순 접근의 한계:
  - 단순히 "더 큰 점수를 주는 계단을 선택한다"는 그리디 방식으로는 나중에 발생할 "3회 연속 금지" 제약을 미리 알 수 없어 최적해를 보장하지 못한다.

- 구조적 특징:
  - `i`번째 계단에 도달했을 때의 최대 점수는 이전 단계들의 최적해를 조합하여 결정되므로 최적 부분 구조(`Optimal Substructure`)를 가진다.

- 선택한 해법:
  - 중복 계산을 방지하기 위해 이전에 계산한 최대 점수를 저장하고 재활용하는 다이나믹 프로그래밍(`DP`)을 사용한다.

<br>

### Approach

---

<!--
- 실제로 채택한 알고리즘
- 자료구조 선택 이유
- 단계별 처리 방식
-->

- 채택한 알고리즘:
  - `Bottom-up DP` (반복문 활용)

- 자료구조 선택:
  - 계단 각각의 점수를 저장할 `scores` 배열과 해당 계단까지의 최댓값을 기록할 `dp` 배열을 사용한다.

- 단계별 처리 방식:
  1. `dp`의 의미 정의:
     - `i`번째 계단에서 얻을 수 있는 점수의 최댓값

  2. 기저 사례(Base Case):
     - 계단이 1개인 경우와 2개인 경우의 최댓값을 미리 정의한다.
       ```
       dp[1] = scores[1]
       dp[2] = scores[1] + scores[2]
       ```
  3. 점화식 설계:
     - `i`번째 계단에 도달하는 경로는 두 가지다.
     - Case 1 (2칸 점프):
       - `i-2`번째에서 바로 `i`번째로 오는 경우.

         ➡️ `dp[i-2] + scores[i]`

     - Case 2 (1칸 점프):
       - `i-1`번째에서 `i`번째로 오는 경우.
       - 이때 연속 3개 제한을 피하려면 `i-1`은 반드시 `i-3`에서 점프해 온 것이어야만 한다.

         ➡️ `dp[i-3] + scores[i-1] + scores[i]`

  4. 최종 계산:
  - `3`번 계단부터 `N`번 계단까지 루프를 돌며 위 두 경우 중 더 큰 값을 `dp[i]`에 저장한다.

<br>

### Complexity

---

- Time complexity: `O(N)`
  - 계단 개수 `N`만큼 한 번의 반복문을 수행하므로 선형 시간에 해결 가능하다.

- Space complexity: `O(N)`
  - 입력 점수와 DP 결과를 저장하기 위한 크기 `N+1`의 배열이 필요하다.

<br>

### Code (Java)

---

```java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        // 계단 개수
        int N = Integer.parseInt(br.readLine());
        // 계단 점수 배열
        int[] scores = new int[N+1];

        for (int i = 1; i <= N; i++) {
            scores[i] = Integer.parseInt(br.readLine());
        }

        if (N == 1) {
            System.out.print(scores[1]);
            return;
        }

        if (N == 2) {
            System.out.print(scores[1] + scores[2]);
            return;
        }

        // i번째 계단에서 얻을 수 있는 점수의 최댓값
        int dp[] = new int[N + 1];
        dp[0] = 0;
        dp[1] = scores[1];
        dp[2] = scores[1] + scores[2];

        for (int i = 3; i <= N; i++) {
            // 점화식
            dp[i] = Math.max(dp[i-2], dp[i-3] + scores[i-1]) + scores[i];
        }

        System.out.print(dp[N]);
    }
}
```
