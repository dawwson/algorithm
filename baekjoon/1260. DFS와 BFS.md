> 풀이 날짜 : 2026.01.27  
> 문제 유형 : 그래프 탐색 (DFS, BFS)  
> 문제 제목 : DFS와 BFS  
> 문제 링크 : https://www.acmicpc.net/problem/1260

### Intuition

- 제약 조건:
  - 정점의 개수 `N(1 <= N <= 1,000)`, 간선의 개수 `M(1 <= M <= 10,000)`

- 핵심 규칙:
  - "방문할 수 있는 정점이 여러 개인 경우에는 정점 번호가 작은 것을 먼저 방문한다"는 조건이 가장 중요하다.

- 선택한 해법:
  - 그래프를 인접 리스트로 구현하여 메모리 효율성을 높였다.
  - 작은 번호부터 방문하기 위해 간선 정보를 입력받은 후 각 리스트를 오름차순 정렬한다.
  - DFS는 재귀(`Recursive`)로, BFS는 큐(`Queue`)를 활용해 구현한다.

### Approach

- 알고리즘:
  - `DFS` (깊이 우선 탐색), `BFS` (너비 우선 탐색)

- 자료구조:
  - `ArrayList<ArrayList<Integer>>`: 간선이 적은 희소 그래프에서 메모리를 아끼고, 연결된 노드만 순회하기에 최적이다.

- 단계별 처리 방식
  - `N+1` 크기의 인접 리스트와 방문 배열을 초기화한다.
  - 양방향 간선 정보를 모두 저장한다 (`u -> v`, `v -> u`)
  - 모든 인접 리스트를 순회하며 `Collections.sort()`로 정렬한다.
  - `DFS`를 수행하며 방문 순서를 기록한다.
  - 방문 배열을 초기화한 후, `BFS`를 수행하며 방문 순서를 기록한다.

### Complexity

- Time complexity

  - 그래프 초기화 및 간선 입력: `O(N + M)`
  - 각 정점의 인접 리스트 정렬: 전체적으로 `O(M log N)`
    - 하나의 정점이 가질 수 있는 최대 인접 정점 수는 `N - 1`이므로 정렬 비용을 `log N` 기준으로 볼 수 있다.
  - DFS: `O(N + M)`
    - 각 정점을 최대 한 번 방문한다.
    - 인접 리스트를 통해 연결된 간선을 확인하므로 전체 간선 확인 비용은 `O(M)`이다.
  - BFS: `O(N + M)`
    - 각 정점을 최대 한 번 큐에 넣고 꺼낸다.
    - 각 정점의 인접 리스트를 순회하므로 전체 간선 확인 비용은 `O(M)`이다.
  - 전체 실행 시간: `O(N + M log N)`
    - DFS와 BFS를 모두 수행해도 `O(N + M) + O(N + M)`은 상수항을 제거하면 `O(N + M)`이다.
    - 여기에 인접 리스트 정렬 비용 `O(M log N)`이 포함되므로 전체 시간복잡도는 `O(N + M log N)`이다.

- Space complexity

  - 인접 리스트에 양방향 간선을 저장하므로 `O(N + M)` 공간이 필요하다.
  - DFS: `O(N)`
    - 방문 배열이 `O(N)`이다.
    - 재귀 호출 스택은 최악의 경우 정점 수만큼 깊어질 수 있으므로 `O(N)`이다.
  - BFS: `O(N)`
    - 방문 배열이 `O(N)`이다.
    - 큐에는 최악의 경우 최대 `N`개의 정점이 들어갈 수 있으므로 `O(N)`이다.
  - 전체 실행 공간: `O(N + M)`
    - 그래프 저장 공간 `O(N + M)`이 가장 크며, DFS/BFS에서 사용하는 추가 공간은 각각 `O(N)`이다.

### Code (Java)

```java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {

    // 인접 리스트
    static ArrayList<ArrayList<Integer>> graph = new ArrayList<>();
    // 방문 여부 체크 리스트
    static ArrayList<Boolean> visited = new ArrayList<>();
    static StringBuilder sb = new StringBuilder();

    public static void main(String[] args) throws Exception {

        // 입출력 스트림 생성
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int N = Integer.parseInt(st.nextToken());
        int M = Integer.parseInt(st.nextToken());
        int V = Integer.parseInt(st.nextToken());

        // 그래프 및 방문 리스트 초기화
        // 정점 번호 1~N을 인덱스로 쓰기 위해 N+1개만큼 만든다.
        for (int i = 0; i <= N; i++) {
            graph.add(new ArrayList<>());
            visited.add(false);
        }

        // 그래프 연결
        for (int i = 0; i < M; i++) {
            st = new StringTokenizer(br.readLine());
            int u = Integer.parseInt(st.nextToken());
            int v = Integer.parseInt(st.nextToken());

            // 양방향 그래프이므로 양쪽에 추가
            graph.get(u).add(v);
            graph.get(v).add(u);
        }

        // 작은 번호부터 방문하기 위해 각 리스트 정렬
        for (int i = 1; i <= N; i++) {
            Collections.sort(graph.get(i));
        }

        dfs(V, visited, graph);
        sb.append("\n");
        Collections.fill(visited, false); // 방문 기록 초기화
        bfs(V, visited, graph);

        System.out.println(sb);
    }

    // DFS - Recursive
    public static void dfs(int current, ArrayList<Boolean> visited, ArrayList<ArrayList<Integer>> graph) {
        // 방문 체크
        visited.set(current, true);

        // 출력
        sb.append(current + " ");

        // 연결된 정점 탐색
        for (int next : graph.get(current)) {
            if (!visited.get(next)) {
                dfs(next, visited, graph);
            }
        }
    }

    // BFS - Queue
    public static void bfs(int start, ArrayList<Boolean> visited, ArrayList<ArrayList<Integer>> graph) {
        Queue<Integer> queue = new LinkedList<>();

        // 큐에 넣고 방문 처리
        queue.add(start);
        visited.set(start, true);

        while (!queue.isEmpty()) {
            int current = queue.poll();
            sb.append(current + " ");

            // 연결된 정점 탐색
            for (int next : graph.get(current)) {
                if (!visited.get(next)) {
                    // 큐에 넣고 방문 처리
                    queue.add(next); // 큐에 넣기
                    visited.set(next, true);
                }
            }
        }
    }
}
```
