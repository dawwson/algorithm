> 문제 유형 : 해시 테이블  
> 문제 제목 : 9375. 패션왕 신해빈  
> 문제 링크 : https://www.acmicpc.net/problem/9375

<br>

### Intuition

---

- 문제의 핵심 요구사항

  - 의상은 여러 종류로 나뉜다.
  - 각 종류에서 최대 1개만 착용 가능하다.
  - 최소 1개 이상의 의상을 착용해야 한다.
  - 가능한 의상 조합의 수를 구하는 문제이다.

- 제약 조건 분석

  - 의상 개수 `0 ≦ n ≦ 30`
  - 테스트 케이스가 여러 개 존재한다.
  - 의상 이름은 중요하지 않고, 종류만 의미를 가진다.

- 필요한 성질

  - 각 종류별 의상 개수만 알면 된다.
  - 조합 계산은 수학적으로 처리할 수 있다.
  - 빠르게 카운팅 할 수 있는 구조가 필요하다.

- 문제풀이 방향
  - 의상 종류별 개수를 해시 테이블로 카운팅한다.
  - 각 종류마다 **(입는 경우 + 안 입는 경우)**가 있고, 모든 종류를 곱한 뒤 **아무것도 안 입은 경우를 제거**한다.

<br>

### Approach

---

- 알고리즘

  - 해시 테이블 + 조합 공식

- 자료구조

  - `HashMap<String, Integer>`
  - `Key`: 의상 종류, `Value`: 해당 종류의 의상 개수

- 처리 단계

  - 각 테스트 케이스마다 의상 개수 `n`을 입력받는다.
  - 의상 정보를 읽으며 `HashMap`에 `{종류 : 개수}` 형태로 카운팅한다.
  - 조합을 계산한다.
    - 각 종류마다 `(개수 + 1)`을 곱한다.
    - `+1`은 해당 종류를 입지 않은 경우를 의미한다.
  - 모든 종류에 대해 곱한 결과에서 아무것도 입지 않은 경우(1)를 제외하여 출력한다.

<br>

### Complexity

---

- Time complexity: `O(n)`

  - 의상 정보 입력 및 카운팅에 `O(n)`, 종류 수만큼 조합 계산에서 `O(n)`
  - 종합적으로 `O(n)`이다.

- Space complexity: `O(n)`

  - 의상 종류별 개수를 저장하기 위해 `Map`을 사용한다.
  - 최악의 경우 모든 의상이 서로 다른 종류일 수 있으므로 `Map`에는 최대 `n`개의 엔트리가 저장된다.

<br>

### Code (Java)

---

```java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {
    public static void main(String[] args) throws Exception {

        // 입출력 스트림 생성
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st;

        // 테스트 케이스 개수
        int tcCount = Integer.parseInt(br.readLine());

        for (int i = 0; i < tcCount; i++) {
            // 의상 개수
            int n = Integer.parseInt(br.readLine());

            // {의상 종류 : 의상 개수}
            // 의상 이름 중복이 없다는 조건 하에 개수만 카운팅
            Map<String, Integer> countMap = new HashMap<>();

            // 의상 정보 저장
            for (int j = 0; j < n; j++) {
                st = new StringTokenizer(br.readLine());
                st.nextToken();

                String type = st.nextToken();
                countMap.put(type, countMap.getOrDefault(type, 0) + 1);
            }

            // 조합 계산하기
            int answer = 1; // 곱셈을 위해 1로 시작

            for (int count : countMap.values()) {
                // (해당 종류 의상 수 + 안 입는 경우 1개)를 곱한다.
                answer *= (count + 1);
            }

            // 출력
            // 아무것도 입지 않은 경우의 수 제외
            System.out.println(answer - 1);
        }
    }
}
```
