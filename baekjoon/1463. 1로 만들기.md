> 문제 제목 : 1463. 1로 만들기  
> 문제 링크 : https://www.acmicpc.net/problem/1463

### Intuition

---

<!--
- 제약 조건
- 단순 접근의 한계
- 필요한 성질 (시간/공간/구조)
- 선택한 해법
-->

- 제약 조건:
  - `N`이 최대 `10^6`이므로, 중복 계산을 피하는 효율적인 알고리즘(`O(N)` 이하)이 필요하다.

- 단순 접근의 한계:
  - 가장 큰 수로 먼저 나누는 그리디 방식은 최적해를 보장하지 않는다.
    (예: `10 ➡️ 5 ➡️ 4 ➡️ 2 ➡️ 1` (4회)보다 `10 ➡️ 9 ➡️ 3 ➡️ 1` (3회)이 더 빠름)

- 필요한 성질:
  - 숫자 `i`를 만드는 최적의 방법은 `i-1, i/2, i/3`을 만드는 최적의 방법들에 의존하므로 최적 부분 구조(`Optimal Substructure`)를 가진다.

- 선택한 해법:
  - 작은 숫자부터 `1`로 만드는 최소 횟수를 기록하며 `N`까지 올라가는 `Bottom-up DP`를 채택한다.

    (`Top-down DP`(재귀) 방식을 사용할 경우, `N`이 커질수록 함수 호출 스택이 깊게 쌓인다.)

<br>

### Approach

---

<!--
- 실제로 채택한 알고리즘
- 자료구조 선택 이유
- 단계별 처리 방식
-->

- 알고리즘:
  - `Bottom-up` 다이나믹 프로그래밍

- 자료구조:
  - 인덱스 자체가 숫자를 의미하고, 해당 숫자의 최소 연산 횟수를 즉시 참조할 수 있도록 1차원 배열(`int[] dp`)을 사용한다.

- 단계별 처리 방식:
  1. 기저 사례 설정: `dp[1] = 0` (`1`은 이미 `1`이므로 연산 횟수 `0`).
  2. 반복문 수행: `2`부터 `N`까지 루프를 돌며 각 숫자의 최솟값을 결정한다.
  3. 비교 로직:
     1. 먼저 `dp[i-1] + 1`을 통해 `1`을 빼는 경우를 기본값으로 잡는다.
     2. 이후 `2`와 `3`으로 나누어떨어지는지 확인하여, `dp[i/2] + 1` 또는 `dp[i/3] + 1` 중 더 작은 값이 있다면 갱신한다.

<br>

### Complexity

---

- Time complexity: `O(N)`
  - 2부터 `N`까지 단 한 번의 루프를 수행하며, 루프 내부 연산은 상수 시간(`O(1)`)이 소요된다.

- Space complexity: `O(N)`
  - `N+1` 크기의 `dp` 배열을 생성하여 모든 숫자에 대한 최적해를 저장한다.

<br>

### Code (Java)

---

```java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        int N = Integer.parseInt(br.readLine());

        if (N == 1) {
            System.out.print(0);
            return;
        }

        // 숫자 i를 1로 만드는 데 필요한 최소 연산 횟수
        int[] dp = new int[N + 1];
        dp[1] = 0;

        for (int i = 2; i <= N; i++) {
            // 1을 빼는 경우 (언제나 가능)
            dp[i] = dp[i-1] + 1;

            // 2로 나누어떨어지는 경우
            if (i % 2 == 0) {
                dp[i] = Math.min(dp[i], dp[i / 2] + 1);
            }

            // 3으로 나누어떨어지는 경우
            if (i % 3 == 0) {
                dp[i] = Math.min(dp[i], dp[i / 3] + 1);
            }
        }

        System.out.print(dp[N]);
    }
}
```
