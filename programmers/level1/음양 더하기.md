### Intuition

---

- `absolutes[i]`는 실제 정수의 절댓값이고, `signs[i]`는 해당 수가 양수인지 음수인지를 나타낸다.
- 두 배열의 같은 인덱스끼리 하나의 숫자를 구성하므로, 인덱스를 기준으로 순회하면 된다.
- `signs[i]`가 `true`이면 더하고, `false`이면 빼면서 누적합을 구한다.

<br>

### Approach

---

1. 정답을 저장할 변수 `answer`를 `0`으로 초기화한다.
2. `absolutes` 배열의 길이만큼 반복한다.
3. 현재 인덱스의 `signs[i]` 값을 확인한다.
4. `signs[i]`가 `true`이면 `absolutes[i]`를 `answer`에 더한다.
5. `signs[i]`가 `false`이면 `absolutes[i]`를 `answer`에서 뺀다.
6. 모든 값을 처리한 뒤 `answer`를 반환한다.

<br>

### Complexity

---

- Time complexity: `O(n)`

- Space complexity: `O(1)`

<br>

### Code (Java)

---

```java
class Solution {
    public int solution(int[] absolutes, boolean[] signs) {
        int answer = 0;
        int l = absolutes.length;

        for (int i = 0; i < l; i++) {
            boolean isPositive = signs[i];

            if (isPositive) {
                answer += absolutes[i];
            } else {
                answer -= absolutes[i];
            }
        }

        return answer;
    }
}
```
