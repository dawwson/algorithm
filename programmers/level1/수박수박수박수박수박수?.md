> 문제 유형 : 연습문제  
> 문제 제목 : 수박수박수박수박수박수?  
> 문제 링크 : https://school.programmers.co.kr/learn/courses/30/lessons/12922

<br>

### Intuition

---

- 길이가 `n`인 문자열을 만들어야 하며, 문자는 `"수"`, `"박"`이 번갈아 나온다.
- 즉, 두 글자 패턴인 `"수박"`을 반복하면 대부분의 문자열을 만들 수 있다.
- `n`이 짝수라면 `"수박"`을 `n / 2`번 반복하면 되고, 홀수라면 마지막에 `"수"`를 하나 더 붙이면 된다.
- 별도의 자료구조나 복잡한 탐색이 필요하지 않은 단순 문자열 생성 문제라고 판단했다.

<br>

### Approach

---

- `n`이 `1`인 경우는 바로 `"수"`를 반환한다.
- `n`이 짝수라면 `"수박".repeat(n / 2)`를 반환한다.
- `n`이 홀수라면 `"수박".repeat(n / 2)`로 만들 수 있는 짝수 길이 부분을 먼저 만들고, 마지막에 `"수"`를 붙인다.
- Java의 `String.repeat()`을 사용해 반복 문자열 생성을 간단하게 처리한다.

<br>

### Complexity

---

- Time complexity: `O(n)`
  - 길이 `n`의 결과 문자열을 생성해야 하므로 문자열 생성 비용은 `O(n)`이다.

- Space complexity: `O(n)`
  - 반환할 문자열의 길이가 `n`이므로 결과 문자열 저장 공간이 `O(n)`이다.

<br>

### Code (Java)

---

```java
class Solution {
    public String solution(int n) {
        if (n == 1) {
            return "수";
        }

        if (n % 2 == 0) {
            return "수박".repeat(n/2);
        } else {
            return "수박".repeat(n/2) + "수";
        }
    }
}
```
