### Intuition

---

- 입력 문자열은 `점수 + 보너스 + 옵션` 형태의 다트 결과가 3번 이어진 구조이다.
- 점수는 `0`부터 `10`까지 가능하므로 한 글자씩 읽을 때 `10`을 처리할 수 있어야 한다.
- `S`, `D`, `T`는 각각 점수의 `1`, `2`, `3`제곱을 의미한다.
- 옵션 `*`는 현재 점수와 바로 이전 점수를 2배로 만들고, `#`은 현재 점수를 음수로 만든다.
- 따라서 문자열을 앞에서부터 순회하며 각 회차의 점수를 계산하고, 옵션이 나오면 이미 계산된 점수를 수정하면 된다.

<br>

### Approach

---

- `rules` Map에 보너스 문자별 제곱 값을 저장한다.
- 문자열을 한 글자씩 순회한다.
- 숫자가 나오면 `score` 문자열에 누적한다.
  - 점수 `10`이 두 글자로 들어올 수 있으므로 바로 숫자로 변환하지 않고 누적한다.
- `S`, `D`, `T`가 나오면 지금까지 누적한 `score`를 정수로 바꾼 뒤 해당 보너스만큼 제곱한다.
- 계산된 점수를 `sets` 배열의 현재 위치에 저장하고 다음 회차로 이동한다.
- 옵션 문자가 나오면 마지막으로 저장한 점수를 수정한다.
  - `*`이면 현재 점수를 2배로 만들고, 이전 점수가 있다면 이전 점수도 2배로 만든다.
  - `#`이면 현재 점수의 부호를 반대로 바꾼다.
- 세 회차의 점수를 모두 더해 반환한다.

<br>

### Complexity

---

- Time complexity: `O(n)`
  - `n`은 `dartResult`의 길이이다.
  - 문자열을 한 번만 순회하므로 시간 복잡도는 `O(n)`이다.

- Space complexity: `O(1)`
  - 다트는 항상 3회만 던지므로 점수 배열의 크기는 입력 크기와 무관하게 고정이다.

<br>

### Code (Java)

---

```java
import java.util.*;

class Solution {
    public int solution(String dartResult) {

        Map<Character, Integer> rules = Map.of(
            'S', 1,
            'D', 2,
            'T', 3
        );

        String score = "";
        int[] sets = new int[3];
        int l = 0;

        for (char ch : dartResult.toCharArray()) {

            if (Character.isDigit(ch)) {
                score += ch;
                continue;
            }

            boolean isArea = "SDT".indexOf(ch) != -1;

            if (isArea) {
                int base = Integer.parseInt(score);
                int exponent = rules.get(ch);
                int set = (int) Math.pow(base, exponent);

                sets[l] = set;
                l += 1;
                score = "";
                continue;
            }

            boolean isOption = "*#".indexOf(ch) != -1;

            if (isOption) {
                if (ch == '*') {
                    sets[l-1] *= 2;
                    if (l > 1) {
                        sets[l-2] *= 2;
                    }
                }

                if (ch == '#') {
                    sets[l - 1] *= -1;
                }
            }
        }

        return sets[0] + sets[1] + sets[2];
    }
}
```

### Better Code (Java)

---

```java
class Solution {
    public int solution(String dartResult) {
        int[] scores = new int[3];
        int index = -1;

        for (int i = 0; i < dartResult.length(); i++) {
            char ch = dartResult.charAt(i);

            if (Character.isDigit(ch)) {
                index++;

                if (ch == '1' && i + 1 < dartResult.length() && dartResult.charAt(i + 1) == '0') {
                    scores[index] = 10;
                    i++;
                } else {
                    scores[index] = ch - '0';
                }
                continue;
            }

            if (ch == 'D') {
                scores[index] *= scores[index];
                continue;
            }

            if (ch == 'T') {
                scores[index] *= scores[index] * scores[index];
                continue;
            }

            if (ch == '*') {
                scores[index] *= 2;

                if (index > 0) {
                    scores[index - 1] *= 2;
                }
                continue;
            }

            if (ch == '#') {
                scores[index] *= -1;
            }
        }

        return scores[0] + scores[1] + scores[2];
    }
}
```

- `score` 문자열을 계속 이어 붙이고 `Integer.parseInt()`로 변환하지 않고, 점수를 만나는 즉시 숫자로 저장한다.
- 점수 `10`만 예외적으로 다음 문자를 확인해 처리한다.
- `S`는 1제곱이므로 별도 연산 없이 그대로 둔다.
- `Math.pow()`는 `double` 기반 연산이므로 이 문제처럼 제곱 수가 작고 정해져 있을 때는 직접 곱셈으로 처리할 수 있다.
- 시간 복잡도는 `O(n)`, 공간 복잡도는 `O(1)`이다.

### Code (Python)

---

```python
def solution(dartResult):
    rules = {'S': 1, 'D': 2, 'T': 3}
    results = []

    score = ''

    for c in dartResult:
        if c.isdigit():
            score += c
            continue
        if c in rules:
            results.append(int(score) ** rules[c])
            score = ''
            continue
        if c == '*':
            if len(results) == 1:
                results[0] *= 2
                continue
            if len(results) > 1:
                results[-1] *= 2
                results[-2] *= 2
                continue
        if c == '#':
            results[-1] *= -1
            continue

    return sum(results)
```
